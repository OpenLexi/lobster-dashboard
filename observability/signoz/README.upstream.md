# OpenClaw-Overview Dashboard

The OpenClaw Overview dashboard gives you a single-pane view of LLM token usage, queue and session health, and error logs so you can quickly spot cost spikes, and stuck sessions. Follow [Import Dashboard Guide](https://signoz.io/docs/dashboards/import-dashboard/) to import this JSON in SigNoz Cloud.

- Use `openclaw-monitoring-with-opentelemetry.json` for a OpenClaw-Overview dashboard.
